# Overview
The following web application called as Complaint management system it allows users to submit and manage complaints 
The system provides role-based access where normal users can manage their own complaints, and admin can manage all users and complaints.

# Used
- Frontend: Next.js  
- Backend: Node.js  
- Database: MySQL  

# Features
- User registration and login  
- Complaint submission, viewing, and deletion  
- Role-based access control (Admin and User)  
- Admin panel for managing users and complaints  
- Restriction on deleting resolved complaints  

# Security Features
- Password hashing using bcrypt  
- Session-based authentication with cookies  
- Role-based authorization for admin and user access  
- Input validation to prevent invalid or malicious data  
- Middleware for session and access control  
- Use of prepared statements and ORM to reduce SQL injection risks  
- Secure API handling and error handling   

# Setup Instructions
1. Clone the repository  
2. Install dependencies using npm install  
3. Configure database connection  
4. Run the backend server  
5. Run the frontend application  

# Usage
- Register a new account or log in with existing credentials  
- Submit complaints and manage them through the dashboard  
- Admin users can manage users and complaints through the admin panel
